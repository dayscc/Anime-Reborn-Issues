# Anime-Reborn-Issues
This repository is used for tracking issues related to Anime Reborn.

Please report:
- Bugs
- UI issues
- Mobile issues
- Performance issues

Use the matching GitHub issue template when opening a report.

Quick guide:
- `Bug report`: something is broken or behaving incorrectly.
- `UI Issue`: layout, visuals, text, overlap, missing assets, or incorrect interface behavior.
- `Mobile Issue`: touch controls, scaling, mobile-only layout, or device-specific problems.
- `Performance Issue`: lag, FPS drops, freezes, stutters, crashes, or slow loading.
- `Enhancement`: a small improvement request for an existing feature.

What makes a report useful:
- One issue per report.
- Clear steps to reproduce.
- Expected result and actual result.
- Device or platform used when relevant.
- Screenshot or video if it helps explain the issue.

Before submitting, check existing issues to avoid duplicates.
